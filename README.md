# PCF Pellets

Веб-застосунок для розрахунку **Product Carbon Footprint (PCF)** при виробництві
залізорудних окатишів. Орієнтований на інженерів-екологістів ГЗК Кривого Рогу
(Полтавський ГЗК, ЦГЗК) для квартальних **CBAM**-звітів
та повних **ISO 14067** звітів про вуглецевий слід продукту.


# Запустити backend + frontend паралельно
npm run dev
